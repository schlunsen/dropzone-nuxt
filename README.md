# nuxt-dropzone

A module for Nuxt.js.

## Installation

```bash
$ npm i nuxt-dropzone 
```

## Setup
- Add `nuxt-dropzone` dependency using yarn or npm to your project
- Add `nuxt-dropzone` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-dropzone',

    // With options
    ['nuxt-dropzone', { /* module options */ }],
 ]
}
```
